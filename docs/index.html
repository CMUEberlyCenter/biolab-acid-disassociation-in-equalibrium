Deployment

Clean out old files and generate a production bundle:

npm run clean
npm run build

ESLint must succeed or the build will fail.

The index.html and bundle.js in ./dist should be placed in the same directory. They can be served by any standard webserver or index.html can be opened by a browser using File > Open File....
